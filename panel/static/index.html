使用帮助

快速开始

  1. 将目标设备通过 USB 连接到电脑
  2. 在有线模式下点击「扫描设备」检测设备
  3. 部署面板内核 (xwebd) 到设备
  4. 部署语音助手 (sair) 到设备
  5. 切换至无线模式，输入设备 IP 连接，获得完整控制功能

两种模式

  • 有线模式 — 通过 USB/ADB 连接设备，适用于首次部署和基础操作（部署内核、部署助手、重启/关机、查看日志）
  • 无线模式 — 通过 WiFi/HTTP 连接设备端 xwebd，提供完整控制功能（音量/亮度、文件管理、配置修改、自检等）

注意事项

  • 部署固件时请勿断开设备连接
  • 重启设备后需要等待约 30 秒才能重新连接
  • 删除设备文件时请谨慎操作，系统文件不可删除
