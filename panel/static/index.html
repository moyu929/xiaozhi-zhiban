使用帮助

快速开始

  1. 将目标设备通过 USB 连接到电脑
  2. 在设备状态卡片中输入设备 IP 地址，点击「连接」
  3. 首次使用时，点击「部署 xwebd」将控制守护进程安装到设备
  4. 部署 Assistant 固件，设备即可使用语音助手功能

功能说明

  • 设备状态 — 实时显示设备型号、系统信息、网络、电量等
  • 设备控制 — 调节音量/亮度、唤醒设备、中止对话、重启/关机
  • Assistant 管理 — 部署/更新/卸载语音助手，配置WebSocket/唤醒词/灵敏度
  • xwebd 管理 — 更新/重启/卸载设备端守护进程，配置日志级别
  • 服务状态 — 查看系统服务运行状态（Telnet/看门狗/自启动等）
  • 设备自检 — 检测设备环境兼容性，诊断各模块是否正常
  • 文件管理 — 浏览设备文件系统、上传/下载/删除文件
  • 日志 — 查看设备实时日志，支持按来源和级别过滤

连接方式

  • 网络连接 — 输入设备 IP 地址（设备与电脑需在同一局域网）
  • ADB 连接 — 通过 USB 连接设备后，系统会自动检测

注意事项

  • 部署固件时请勿断开设备连接
  • 重启设备后需要等待约 30 秒才能重新连接
  • 删除设备文件时请谨慎操作，系统文件不可删除
