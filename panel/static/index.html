使用帮助

快速开始

  1. 将目标设备通过 USB 连接到电脑
  2. 在有线模式下点击「扫描设备」检测设备
  3. 运行环境自检，确认设备满足部署条件
  4. 依次部署面板内核 (xwebd) 和语音助手 (sair)
  5. 切换至无线模式，输入设备 IP 连接，获得完整控制功能

两种模式

  • 有线模式 — 通过 USB/ADB 连接设备，适用于首次部署和基础操作（部署内核、部署助手、重启/关机、查看日志）
  • 无线模式 — 通过 WiFi 连接设备端 xwebd 内核，提供完整控制功能（配置修改、MCP管理、文件管理、自检、日志查看等）

三模块架构

  • Panel — 运行在电脑上的 Web 控制面板（Python），端口 3000
  • xwebd — 运行在设备上的 HTTP 服务（C），端口 8080，作为 Panel 与设备之间的桥梁
  • sair (语音助手) — 运行在设备上的语音助手进程（C），通过文件 IPC 与 xwebd 通信

MCP 协议

  • 语音助手通过 MCP (Model Context Protocol) 协议控制设备硬件（音量、亮度、重启等）
  • MCP 消息通过 WebSocket 连接传输，接入点地址可在「服务状态 → MCP 可视化管理」中配置
  • 不同设备的 MCP 接入点可能不同，首次使用需手动配置正确的 WebSocket 地址

更新方式

  • 部署 — 全新安装或替换原生助手，设备会重启
  • 冷更新 — 替换已安装的自定义助手，设备会重启
  • 热更新 — 在助手运行中无缝替换，设备不重启（仅自定义助手支持）

注意事项

  • 部署固件时请勿断开设备连接
  • 重启设备后需要等待约 30 秒才能重新连接
  • 删除设备文件时请谨慎操作，系统文件不可删除
  • 设备音量/亮度由语音助手通过 MCP 协议控制，无需手动调节
